Java Web Development Boot Camp



Description

Java web development bootcamp is a remote learning program. It helps students learn how to
develop complete web applications in the java language with Oracle as the database.By the end

of the course, each student will have to develop two complete web applications: an online messaging

app like Twitter and an online survey tool. This course will cover: Servlets, JSP,Expression Language,

JSTL, JPQL, Sessions/Cookies, HTML/CSS/Javascript and the Bootstrap Framework.

Prerequisites

Programming knowledge with a basic understanding of an object oriented language: Java, C#,

VB.NET or C++. Experience developing web applications is not required.

Objectives

  • Develop complete web applications in the Java language with Oracle as the database.
  • Program with Java EE framework and associated technologies.
  • Develop an online messaging app like Twitter.
  • Develop and online survey tool.
